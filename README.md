# Formação C# — Rocketseat 🚀

Este repositório reúne **toda a minha jornada de aprendizado em C# pela Rocketseat**, concentrando **todos os projetos desenvolvidos ao longo da formação em um único lugar**.

A ideia aqui é simples: **um repositório único para a formação**, organizado por módulos, facilitando a evolução, revisão de conteúdo e acompanhamento do meu progresso.

---

## 📂 Organização do Repositório

Cada pasta representa um **módulo, aula ou tema específico** abordado durante a formação.

Exemplos de organização:

```
📁 CashFlow       
📁 MyFirstApi     
📁 Petfolio       
```

* Cada pasta é **independente**
* Pode conter um ou mais projetos
* Reflete exatamente o conteúdo estudado naquele módulo
* Todos os projetos adotam os príncipios do **Domain-Driven Design (DDD)**

---

## 🎯 Objetivo

Este repositório tem como objetivo:

* Consolidar todo o aprendizado da formação em C#
* Servir como **material de estudo e revisão**
* Demonstrar evolução técnica ao longo do tempo
* Manter um histórico real de aprendizado (commits, refactors e melhorias)

👉 **Não serão criados novos repositórios para projetos da formação**.</br>
👉 Projetos em repositórios separados existirão **apenas para projetos práticos, desafios ou exercícios independentes**.

---
